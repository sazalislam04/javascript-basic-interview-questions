Double Equal and Triple Equal: Double Equals checks for value equality only. It inherently does type coercion. This means that before checking the values. it converts the type of the variables to match each other.
Triple Equals: on the other hand, Triple Equals does not perform type coercion. It will verify whether the variables being compared have both the same value and the same type.
code exmaple:
const number = 1234 const stringNumber = '1234' console.log(number == stringNumber) //true console.log(number === stringNumber) //false
